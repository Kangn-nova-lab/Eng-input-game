# English Typing Practice Game

## Overview

This project is an English typing practice game. Users can type the words on the screen to test their typing speed and accuracy.

## Features

*   **Intro Screen:** A starting screen with a 'Start Game' button.
*   **Pause/Resume:** A button to pause and resume the game.
*   **Falling Words:** Random English words fall from the top of the screen.
*   **Typing Input:** Users can type the words in an input field and press Enter to make them explode.
*   **Speed Control:** A slider allows the user to control the speed of the falling words (levels 1-5).
*   **Explosion Effect:** Correctly typed words explode with a visual effect.
*   **Timer:** A timer tracks the time taken to type the words.
*   **Score:** The user's score is calculated based on their typing speed and accuracy.
*   **Game Over:** The game ends when the timer runs out.
*   **Visuals**: A dynamic, animated gradient background and custom fonts.

## Project Structure

*   `index.html`: The main HTML file for the game.
*   `style.css`: The CSS file for styling the game.
*   `main.js`: The JavaScript file for the game logic.
